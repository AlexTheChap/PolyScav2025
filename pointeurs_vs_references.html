Pointeurs vs Références (C++)

Ce site explique la différence entre un pointeur et une référence en C++, en s’inspirant des concepts du cours INF1015

1. Définitions

Pointeur

Un pointeur est une variable qui contient l’adresse mémoire d’un autre objet. Il peut changer de cible, être nul (nullptr) et inclut une opération de déréférencement (*).

int x = 10;
int* p = &x; // p pointe vers x
*p = 20;     // modifie x

Référence

Une référence est un alias vers une variable existante. Une fois initialisée, elle ne peut plus changer de cible contrairement à un pointeur. 

int y = 10;
int& r = y; // r est une référence vers y
r = 20;     // modifie y

4. Quand et quoi utiliser ?

Utiliser une référence si :

  • La variable doit toujours être valide.
  • Simplifier la syntaxe.

Utiliser un pointeur si :

  • La cible peut changer durant l’exécution.
  • La variable peut parfois être null (nullptr).
  • Utilisation l’allocation dynamique.

5. Exemple combiné

int a = 5;
int b = 9;
int* p = &a; // pointeur
int& r = b; // référence

*p = 12; // modifie a
r = 15;  // modifie b

p = &b;  // p change de cible (ok)
// r = a; impossible : référence figée
